# HR_Analyst_Dashboard
This HR Analytics Dashboard provides a clear overview of employee distribution, attrition, and job satisfaction trends. It visualizes key metrics such as department-wise attrition, age groups, education fields, and gender-based insights. The dark purple and maroon theme enhances readability while presenting workforce patterns at a glance.

#Description
This HR Analytics Dashboard provides a visual overview of employee demographics, attrition patterns, and job satisfaction insights using a clean dark purple and maroon theme.

#Key Performance Indicators
-Total Employees
-Active Employees
-Attrition
-Attrition Rate

#Objectives
-Understand workforce composition
-Identify attrition patterns
-Analyze job satisfaction across roles
-Explore demographic insights across age, gender, and education fields
